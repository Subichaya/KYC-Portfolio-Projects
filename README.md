# KYC AML Risk Analytics Portfolio

## About
Portfolio demonstrating KYC/AML compliance analytics skills 
built by a compliance data analyst with 7+ years experience.

## Projects

### 1. SQL Customer Risk Scoring
- Built KYC risk scoring engine in MySQL
- Identifies high risk countries, large transactions, 
  PEPs and sanctions matches
- Assigns HIGH/MEDIUM/LOW risk ratings

### 2. Python Compliance Automation
- Automated KYC compliance report generation
- Applies FATF-based risk scoring rules
- Outputs colour coded Excel report automatically

### 3. Power BI KYC Dashboard
- Interactive dashboard with 4 visuals + 3 KPI cards
- Shows risk distribution, transaction analysis,
  customer flags and individual risk scores
